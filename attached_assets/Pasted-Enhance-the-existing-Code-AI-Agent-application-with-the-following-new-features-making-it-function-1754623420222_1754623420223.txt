Enhance the existing "Code AI Agent" application with the following new features, making it function more like Replit AI but using the Gemini AI API:

1. GitHub Repository Integration:

Allow the user to connect their GitHub account using OAuth 2.0 GitHub App integration.

After connection, display a list of the user’s repositories.

Enable full Git functionality inside the app:

Clone a repo into the app’s workspace.

Create/edit/delete files.

Commit changes with a message.

Push changes to GitHub.

Create new branches, switch branches, and merge branches.

View Git commit history and diffs.

2. AI-Powered Code Editing:

AI should be able to read repo code, suggest changes, and apply updates directly to the files.

Support AI-assisted pull request creation with commit messages generated by Gemini.

3. Full App Download:

Add a "Download Full App" button to let the user download the entire workspace/project as a .zip file.

The .zip should include all generated or modified files along with a README.md.

4. Frontend Updates:

Add a sidebar file explorer showing all project files.

Implement an integrated code editor (Monaco Editor or CodeMirror) with syntax highlighting and auto-complete.

Enable drag-and-drop file uploads into the workspace.

Show Git status in the UI (modified, untracked, committed).

5. Backend Updates:

Add endpoints for:

GitHub OAuth authentication and token storage (in session).

Git operations (clone, commit, push, pull, branch, merge).

AI-assisted file modifications.

Zipping the entire project for download.

Use Node.js + simple-git (or Python GitPython) for Git operations.

6. AI Features:

Allow the user to select a file or repo folder and ask the AI to:

Refactor code.

Add features.

Fix bugs.

Write documentation.

AI should update the files automatically and show a Git diff preview before committing.

7. Deployment:

Ensure GitHub OAuth works on deployed version.

Keep user repo access private and secure.

Goal: Transform the app from a simple Gemini AI coding chatbot into a full AI-powered GitHub IDE with real-time coding, Git integration, and full project download.
